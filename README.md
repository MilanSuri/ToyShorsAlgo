# ToyShorsAlgo
RSA Factoring Demo with Shor's Algorithm (Toy Simulation)

**Author:** Milan Suri  
**Course:** CSC 471  
**Date:** November 10, 2025  

## Features

- Simulates **Shor’s algorithm** using Qiskit Aer simulator.
- Demonstrates **Quantum Phase Estimation (QPE)** and **Quantum Fourier Transform (QFT)**.
- Interactive **Streamlit UI** to factor small RSA numbers.
- Includes **automated test cases** to verify correctness.
- Educational explanations of Shor’s algorithm, quantum concepts, and classical post-processing.
